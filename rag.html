RAG Pipeline

1. User Query

The user asks a question.

2. Retrieval Engine

The system searches the knowledge base for relevant documents.

3. Ranking & Retrieval

Relevant chunks are ranked based on similarity.

4. Context Injection

The retrieved chunks are added into the LLM prompt.

5. Response Generation

The LLM generates an answer using retrieved context.
